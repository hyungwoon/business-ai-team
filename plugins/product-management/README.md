# Product Management Plugin

> 제품 관리 종합 플러그인 — 60개 스킬, 31개 커맨드

## 개요

PM 전체 라이프사이클을 지원하는 종합 플러그인입니다. Anthropic의 기본 6개 스킬에 PM Skills (Paweł Huryn)의 57개 스킬을 통합하여, 디스커버리부터 성장까지 모든 단계를 커버합니다.

- **60개 스킬** — 8개 카테고리로 체계화
- **31개 커맨드** — 7개 pm-skills-ko 플러그인에서 제공
- **12개 MCP 커넥터** — Slack, Linear, Notion, Figma, Amplitude 등 연동

## 스킬 목록 (60개)

### 디스커버리 (12)

| 스킬 | 설명 |
|------|------|
| `user-research-synthesis` | 인터뷰, 설문, 지원 티켓을 구조화된 인사이트로 변환. 테마 식별, 페르소나 개발, 기회 영역 도출 |
| `user-interview-guide` | 효과적인 사용자 인터뷰 계획 및 진행. 질문 설계, 인터뷰 진행 방법, 데이터 분석 |
| `survey-design` | 정량적 데이터 수집을 위한 설문 설계. 질문 유형, 샘플 크기, 분석 방법 |
| `user-personas` | 사용자 페르소나 개발 및 검증. 데이터 기반 페르소나, 행동 패턴, 동기 분석 |
| `jobs-to-be-done` | Jobs to be Done 프레임워크 적용. 고객 목표, 상황, 해결책 매핑 |
| `empathy-mapping` | 공감 맵 작성으로 사용자 이해 심화. 생각, 느낌, 행동, 통증점 분석 |
| `user-journey-mapping` | 사용자 여정 맵 작성. 터치포인트, 감정 곡선, 개선 기회 식별 |
| `competitive-analysis` | 경쟁사 분석 및 포지셔닝. 기능 비교, 강점/약점, 전략적 함의 |
| `market-research` | 시장 규모, 성장률, 트렌드 조사. 1차/2차 조사 방법론 |
| `trend-analysis` | 산업 트렌드 분석 및 미래 예측. 신호 감지, 영향 평가 |
| `customer-feedback-analysis` | 고객 피드백 수집 및 분석. 피드백 루프, 우선순위 지정 |
| `problem-validation` | 문제 가설 검증. 검증 방법, 성공 기준, 학습 계획 |

## MCP 커넥터

다음 도구와 연동하여 더욱 강력한 기능을 제공합니다:

- **Slack** — 팀 맥락 및 이해관계자 스레드
- **Linear** — 로드맵 통합, 티켓 맥락, 상태 추적
- **Asana** — 프로젝트 관리, 작업 추적
- **monday.com** — 워크플로우 자동화, 상태 관리
- **ClickUp** — 작업 관리, 타임라인
- **Atlassian** — Jira 통합, 이슈 추적
- **Notion** — 기존 스펙, 리서치, 회의록
- **Figma** — 디자인 맥락, 핸드오프
- **Amplitude** — 사용자 데이터, 지표, 행동 분석
- **Pendo** — 제품 분석, 사용자 피드백
- **Intercom** — 지원 티켓, 기능 요청, 사용자 대화
- **Fireflies** — 회의 기록, 토론 맥락

## 사용 예시

### 기능 스펙 작성

```
당신: /write-spec
Claude: 어떤 기능이나 문제를 스펙으로 작성하고 싶으신가요?
당신: 엔터프라이즈 고객을 위해 SSO 지원을 추가해야 합니다
Claude: [대상 사용자, 제약사항, 성공 지표에 대해 질문]
Claude: [문제 정의, 사용자 스토리, 요구사항, 성공 지표, 미해결 질문이 포함된 완전한 PRD 생성]
```

### 이해관계자 업데이트 준비

```
당신: /stakeholder-update
Claude: 어떤 유형의 업데이트인가요? (주간, 월간, 출시, 기타)
당신: 경영진 팀을 위한 주간 업데이트
Claude: [프로젝트 관리 도구, 채팅, 문서에서 맥락 수집]
Claude: [진행 상황, 의사결정, 위험, 다음 마일스톤이 포함된 경영진 요약 생성]
```

### 사용자 리서치 종합

```
당신: /synthesize-research
Claude: 어떤 리서치를 종합하고 싶으신가요? 인터뷰 노트를 붙여넣거나 파일을 업로드하거나 연결된 소스에서 가져올 수 있습니다.
당신: [8개의 인터뷰 기록 붙여넣기]
Claude: [인터뷰 전체에서 테마, 패턴, 인사이트 식별]
Claude: [주요 발견, 페르소나, 기회 영역, 증거가 포함된 종합 보고서 생성]
```

### 경쟁사 분석

```
당신: /competitive-brief
Claude: 어떤 경쟁사나 기능 영역을 분석하고 싶으신가요?
당신: 우리의 온보딩 플로우를 [경쟁사 A]와 [경쟁사 B]와 비교해주세요
Claude: [경쟁사 온보딩 접근 방식 조사]
Claude: [기능 비교, 강점/약점, 전략적 함의가 포함된 브리프 생성]
```
