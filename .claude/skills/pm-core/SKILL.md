# 핵심 PM 운영 (Core PM)

사용자 요청에 따라 아래 프레임워크 중 적절한 것을 선택하여 적용합니다.
`knowledge/product.md`가 존재하면 먼저 읽어 보정 사항을 반영합니다.

## 스킬 라우팅

| 요청 유형 | 참조 파일 |
|---|---|
| 로드맵 관리/우선순위화 | [references/roadmap-management.md] |
| 제품 지표 정의/추적/분석 | [references/metrics-tracking.md] |
| 이해관계자 업데이트/커뮤니케이션 | [references/stakeholder-comms.md] |

## 사용법

1. 사용자의 요청을 위 라우팅 테이블에서 매칭
2. 해당 `references/` 파일을 Read
3. 파일의 프레임워크와 지시사항에 따라 실행
4. 결과물은 마크다운 문서로 저장
