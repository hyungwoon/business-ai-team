# 경쟁사 분석

경쟁 포지셔닝 및 시장 환경 매핑 전문 경쟁 인텔리전스 스킬. 경쟁사 분석, 기능 비교, 포지셔닝 평가, 전략적 시사점 도출을 지원합니다.

## 1. 경쟁 환경 매핑

### 경쟁사 집합 정의

경쟁사를 여러 레벨로 구분합니다:

**직접 경쟁사**: 동일한 사용자의 동일한 문제를 동일한 방식으로 해결하는 제품.
- 고객이 실제로 비교 평가하는 제품
- 딜 과정, 고객 비교, 리뷰 사이트 매치업에 등장

**간접 경쟁사**: 동일한 문제를 다른 방식으로 해결하는 제품.
- 동일한 사용자 니즈에 다른 접근법 (예: 스프레드시트 vs 전용 프로젝트 관리 도구)
- "비소비(non-consumption)" 포함 — 아무것도 하지 않거나 수동 프로세스를 사용하는 경우

**인접 경쟁사**: 현재는 경쟁하지 않지만 잠재적으로 경쟁할 수 있는 제품.
- 유사한 기술, 고객 기반, 유통망을 가진 기업
- 기능을 추가할 수 있는 대형 플랫폼
- 틈새 시장을 공략하다 핵심 시장으로 성장할 수 있는 스타트업

**대체 솔루션**: 사용자가 근본적인 니즈를 해결하는 완전히 다른 방법.
- 소프트웨어 구매 대신 사람 고용
- 전문 도구 대신 범용 도구 (Excel, 이메일) 사용
- 프로세스 전체 아웃소싱

### 경쟁 환경 지도

의미 있는 축으로 경쟁사를 포지셔닝합니다:

**주요 축 예시**:
- 폭 vs 깊이 (스위트 vs 포인트 솔루션)
- SMB vs 엔터프라이즈 (시장 세그먼트 초점)
- 셀프서브 vs 영업 주도 (GTM 방식)
- 단순 vs 강력 (제품 복잡도)
- 수평적 vs 수직적 (범용 vs 산업 특화)

전략적 포지셔닝 차이를 드러내는 축을 선택하세요. 올바른 축은 경쟁 역학을 가시화합니다.

### 경쟁 환경 모니터링

시간에 따른 경쟁 움직임을 추적합니다:
- 제품 출시 및 기능 릴리스 (체인지로그, 블로그, 보도자료)
- 가격 및 패키징 변경
- 투자 유치 및 인수
- 핵심 인재 채용 및 채용 공고 (전략적 방향 신호)
- 고객 획득 및 이탈 (특히 자사의 Win/Loss)
- 애널리스트 및 리뷰 커버리지
- 파트너십 발표

## 2. 경쟁사 프로파일링

5개의 주요 직접 경쟁사 각각에 대해 다음 구조로 분석합니다:

**경쟁사 프로필**
- 회사명, 설립일, 투자 유치/현황
- 주요 시장 초점 및 서비스 고객 세그먼트
- 추정 시장 점유율 또는 고객 기반 규모
- 시장 포지셔닝 및 GTM 전략

**핵심 제품 강점**
- 주요 기능 및 역량
- 고유한 경쟁 우위
- 고객 가치 제안
- 기술 차별화 또는 해자
- 고객 만족도 및 유지율 신호

**제품 약점 & 격차**
- 누락된 기능 또는 사용 사례
- 알려진 한계 또는 고객 페인 포인트
- 기술적 또는 운영상 약점
- 시장 포지셔닝 격차
- 고객 불만족 영역

**비즈니스 모델 & 가격**
- 가격 구조 (좌석당, 사용량당, 정액제, 프리미엄 등)
- 시장 내 가격대
- GTM 채널 및 영업 방식
- 수익 모델 및 성장 단계

**경쟁 위협 & 이점**
- 이 경쟁사가 자사에 미치는 위협
- 기존 고객 기반 및 전환 비용
- 전략적 파트너십 또는 에코시스템
- 최근 제품 업데이트 또는 전략적 변화

## 3. 기능 비교 매트릭스

### 비교 매트릭스 구성 방법

1. **역량 영역 정의**: 기능을 구매자가 평가할 때 사용하는 기능 카테고리로 묶습니다 (내부 아키텍처 기준이 아닌 구매자 기준).
2. **세부 기능 나열**: 각 영역 아래 비교할 구체적인 기능이나 역량을 나열합니다.
3. **각 경쟁사 평가**: 일관된 평가 척도를 사용합니다.

### 평가 척도

**간단 척도 (대부분의 경우 권장)**:

| 등급 | 의미 |
|------|------|
| 강함 | 시장 선도 역량. 깊은 기능, 잘 실행됨 |
| 적절 | 기능적 역량. 작동하지만 차별화되지 않음 |
| 약함 | 존재하지만 제한적. 중요한 격차 또는 낮은 실행 품질 |
| 없음 | 이 역량 없음 |

**상세 척도 (심층 비교용)**:

| 점수 | 의미 |
|------|------|
| 5 | 동급 최강. 다른 제품이 목표로 삼는 기준 |
| 4 | 강함. 완전한 기능, 잘 실행됨 |
| 3 | 적절. 기본 니즈 충족, 차별화 없음 |
| 2 | 제한적. 존재하지만 중요한 격차 있음 |
| 1 | 최소. 거의 기능하지 않거나 초기 베타 |
| 0 | 없음. 사용 불가 |

### 비교 매트릭스 템플릿

```
| 역량 영역       | 자사 제품 | 경쟁사 A | 경쟁사 B |
|----------------|----------|---------|---------|
| [영역 1]        |          |         |         |
|   [기능 1]      | 강함      | 적절     | 없음     |
|   [기능 2]      | 적절      | 강함     | 약함     |
| [영역 2]        |          |         |         |
|   [기능 3]      | 강함      | 강함     | 적절     |
```

### 기능 비교 팁

- 마케팅 주장이 아닌 실제 제품 경험, 고객 피드백, 리뷰를 기반으로 평가
- 기능은 스펙트럼으로 존재합니다. "기능 X 보유"보다 "기능 X를 얼마나 잘 수행하는가"가 더 유용
- 전체 기능 수가 아닌 타깃 고객에게 중요한 것을 기준으로 비교에 가중치 부여
- 정기적으로 업데이트 — 기능 비교는 빠르게 구식이 됨
- 경쟁사가 앞서는 부분에 솔직하게 평가. 항상 자사가 이기는 비교는 신뢰할 수 없음
- 각 역량 영역에 "왜 중요한가"를 포함. 모든 기능이 구매자에게 동등하게 중요하지 않음

## 4. 포지셔닝 분석

### 포지셔닝 문장 분석

각 경쟁사의 포지셔닝을 추출합니다:

**템플릿**: [타깃 고객]을 위해 [니즈/문제]를 해결하는 [제품]은 [핵심 혜택]을 제공하는 [카테고리]입니다. [경쟁사/대안]과 달리, [제품]은 [핵심 차별화 요소]를 갖추고 있습니다.

**포지셔닝 파악 출처**:
- 홈페이지 헤드라인 및 서브헤드라인
- 앱스토어 또는 리뷰 사이트의 제품 설명
- 영업 피치 덱 (잠재 고객이 공유하는 경우)
- 애널리스트 브리핑 자료
- 실적 발표 언어 (상장사의 경우)

### 메시지 아키텍처 분석

각 경쟁사가 가치를 어떻게 전달하는지 분석합니다:

| 레벨 | 내용 | 예시 |
|------|------|------|
| 레벨 1 — 카테고리 | 어떤 카테고리를 주장하는가? | CRM, 프로젝트 관리, 협업 플랫폼 |
| 레벨 2 — 차별화 요소 | 카테고리 내에서 무엇이 다른가? | AI 기반, 올인원, 개발자 우선 |
| 레벨 3 — 가치 제안 | 어떤 결과를 약속하는가? | 더 빠른 딜 클로징, 더 빠른 제품 출시 |
| 레벨 4 — 증거 | 어떤 증거를 제공하는가? | 고객 로고, 지표, 수상, 케이스 스터디 |

### 포지셔닝 격차 및 기회

다음을 탐색합니다:
- **미점유 포지션**: 구매자에게 중요하지만 어떤 경쟁사도 소유하지 않은 가치 제안
- **혼잡한 포지션**: 모든 경쟁사가 주장하여 의미를 잃은 클레임
- **부상하는 포지션**: 시장 변화(AI, 원격 근무, 컴플라이언스)로 인한 새로운 가치 제안
- **취약한 포지션**: 경쟁사가 주장하지만 완전히 실현하지 못하는 클레임

## 5. Win/Loss 분석

### Win/Loss 분석 수행

Win/Loss 분석은 실제로 딜을 이기고 지는 이유를 밝혀줍니다. 가장 실행 가능한 경쟁 인텔리전스입니다.

**데이터 소스**:
- 영업팀의 CRM 노트 (즉시 사용 가능하지만 편향 있음)
- 결정 직후 고객 인터뷰 (가장 가치 있고 편향 적음)
- 이탈 고객 설문 또는 이탈 인터뷰
- 잠재 고객 설문 (잃은 딜의 경우)

### Win/Loss 인터뷰 질문

**승리한 경우**:
- 어떤 문제를 해결하려 했나요?
- 어떤 대안을 평가했나요? (경쟁 집합 파악)
- 대안 대신 저희를 선택한 이유는 무엇인가요?
- 다른 제품을 선택할 뻔한 순간이 있었나요?
- 저희를 잃으려면 무엇이 필요할까요?

**패배한 경우**:
- 어떤 문제를 해결하려 했나요?
- 결국 무엇을 선택했나요? 이유는?
- 저희 제품이 어디서 부족했나요?
- 저희가 다르게 했다면 어떤 점이 좋았을까요?
- 미래에 저희를 재고할 의향이 있나요? 어떤 조건에서?

### Win/Loss 데이터 분석

- 시간에 따른 Win/Loss 이유 추적. 패턴이 변하고 있나요?
- 딜 유형별 세분화: 엔터프라이즈 vs SMB, 신규 vs 확장, 산업 수직
- 승리 및 패배의 상위 3~5가지 이유 식별
- 제품 이유(기능, 품질)와 비제품 이유(가격, 브랜드, 관계, 타이밍) 구분
- 경쟁사별 경쟁 승률 계산: 각 경쟁사가 포함된 딜에서 몇 %를 이기는가?

### 일반적인 Win/Loss 패턴

| 패턴 | 설명 |
|------|------|
| 기능 격차 | 경쟁사가 딜 브레이커인 특정 역량 보유 |
| 통합 우위 | 경쟁사가 구매자가 이미 사용하는 도구와 통합 |
| 가격 구조 | 항상 더 저렴한 것이 아닌 — 다른 가격 모델이 더 잘 맞는 경우 |
| 기존 업체 우위 | 전환 비용이 너무 높아 구매자가 현재 제품 유지 |
| 영업 실행 | 더 나은 데모, 빠른 응답, 더 관련성 높은 케이스 스터디 |
| 브랜드/신뢰 | 구매자가 더 안전하거나 잘 알려진 옵션 선택 |

## 6. 시장 트렌드 식별

### 트렌드 파악 출처

- **산업 애널리스트 보고서**: Gartner, Forrester, IDC — 시장 규모 및 트렌드
- **벤처 캐피털**: VC가 무엇에 투자하는가? 투자 테마는 스마트 머니가 기회를 보는 곳을 신호
- **컨퍼런스 테마**: 산업 이벤트가 무엇에 집중하는가? 어떤 주제가 가장 많은 청중을 끌어모으는가?
- **기술 변화**: 새로운 플랫폼, API, 역량 — 새로운 제품 카테고리를 가능하게 하는 것
- **규제 변화**: 요구사항이나 기회를 만드는 새로운 규제
- **고객 행동 변화**: 사용자 기대가 어떻게 진화하는가? (모바일 우선, AI 지원, 프라이버시 의식)
- **인재 이동**: 최고 인재들이 어디로 가는가? 어떤 스킬이 수요가 있는가?

### 트렌드 분석 프레임워크

각 식별된 트렌드에 대해:

| 질문 | 내용 |
|------|------|
| 무엇이 변하고 있는가? | 트렌드를 명확하고 구체적으로 설명 |
| 왜 지금인가? | 이 변화를 이끄는 것은? (기술, 규제, 행동, 경제) |
| 누가 영향을 받는가? | 어떤 고객 세그먼트 또는 시장 카테고리? |
| 타임라인은? | 지금 일어나고 있는가, 1~2년 후인가, 3~5년 후인가? |
| 자사에 대한 시사점은? | 이것이 제품 전략에 어떤 영향을 미쳐야 하는가? |
| 경쟁사는 무엇을 하는가? | 경쟁사들이 이 트렌드에 어떻게 대응하는가? |

### 신호 vs 노이즈 구분

- **신호**: 행동 데이터, 증가하는 투자, 규제 조치, 또는 고객 수요로 뒷받침되는 트렌드
- **노이즈**: 미디어 과대 선전, 컨퍼런스 버즈, 또는 고객 트랙션 없는 경쟁사 발표로만 뒷받침되는 트렌드
- 자사 고객 데이터로 트렌드를 검증: 자사 고객들이 이것을 요청하거나 경험하고 있는가?
- "올해의 트렌드" 과대 선전 사이클에 주의. 산업 대화를 지배하는 많은 트렌드가 수년간 고객에게 실질적인 영향을 미치지 않음

### 전략적 대응 옵션

| 대응 | 설명 | 리스크/보상 |
|------|------|------------|
| 선도 | 일찍 투자하여 카테고리나 접근법을 정의 | 높은 리스크, 높은 보상 |
| 빠른 추종 | 고객 수요의 초기 신호를 기다린 후 빠르게 이동 | 낮은 리스크, 차별화 어려움 |
| 모니터링 | 트렌드를 추적하지만 아직 투자하지 않음. 행동 트리거 설정 | 낮은 리스크, 기회 놓칠 수 있음 |
| 무시 | 이 트렌드가 전략과 관련 없다고 명시적으로 결정. 이유 문서화 | 전략적 명확성 |

올바른 대응은 경쟁 포지션, 고객 기반, 자원, 트렌드 속도에 따라 달라집니다.

## 7. 차별화 기회 & 전략적 권고

### 차별화 기회

경쟁사 집합 전반에 걸쳐 다음을 식별합니다:
- 충족되지 않은 고객 니즈
- 차별화를 위한 기능/가격/UX 기회
- 경쟁사가 충분히 서비스하지 못하는 타깃 세그먼트
- 경쟁사가 효과적으로 해결하지 못한 Jobs-to-be-done
- 아직 활용되지 않은 채널 또는 GTM 접근법
- 경쟁사가 부족한 잠재적 파트너십 또는 통합

### 경쟁 포지셔닝 권고

- 권고 경쟁 포지셔닝 및 강조할 핵심 차별화 요소
- 타깃 또는 회피할 세그먼트 또는 사용 사례
- 모니터링할 경쟁 위협
- 향후 12~18개월 경쟁 리스크 및 기회
