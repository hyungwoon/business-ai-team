# 전문가 팀 목록 표시

## 실행 절차

1. `agents/` 폴더의 모든 .md 파일 목록을 Glob으로 확인한다
2. `plugins/` 폴더의 모든 SKILL.md 파일 목록을 Glob으로 확인한다
3. 아래 형식으로 팀 구성을 사용자에게 보여준다

## 출력 형식

```
## 전문가 팀 구성

| 에이전트 | 전문 분야 | 보유 스킬 |
|----------|-----------|-----------|
| Marketing | 마케팅 | brand-voice, content-creation, ... |
| Sales | 영업 | draft-outreach, call-prep, ... |
| ... | ... | ... |

## 사용법

- `/route [비즈니스 요청]` — 전문가 라우팅으로 응답
- `/ask [질문]` — 간편 전문가 질문
- 일반 대화에서도 비즈니스 요청 시 자동 라우팅됩니다
```

4. 각 에이전트 파일의 첫 줄 설명(> 인용문)을 읽어 전문 분야를 채운다
5. 각 에이전트의 플러그인 & 스킬 라우팅 섹션에서 스킬 목록을 추출한다
