# 비즈니스 전문가 질문

사용자 질문: $ARGUMENTS

## 실행 절차

1. 질문의 도메인을 파악하여 해당 `agents/[에이전트].md` 파일을 Read 도구로 읽는다
2. 관련 `plugins/[플러그인]/skills/[스킬]/SKILL.md`를 Read 도구로 읽는다
3. 에이전트의 전문 관점과 스킬의 베스트 프랙티스를 반영하여 답변한다
4. 응답 맨 앞에 `> **담당**: [에이전트] | 참조 스킬: [스킬]` 표시

**에이전트/스킬 파일을 읽지 않고 응답하는 것은 금지된다.**
